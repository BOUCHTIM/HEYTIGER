# Hey, Tiger — Font Files

Drop your licensed font files here. The CSS is already wired up.

## Required fonts

### Display — TAN Nimbus
- `TAN-Nimbus.woff2`
- `TAN-Nimbus.woff`  (fallback)
Purchase: https://tan.typenetwork.com

### Body — JANDUS RD. 20
- `JandusRD20-Regular.woff2`
- `JandusRD20-Regular.woff`
- `JandusRD20-Bold.woff2`
- `JandusRD20-Bold.woff`

Until the files are present the page uses:
- Display fallback → Fraunces 900 (Google Fonts — closest psychedelic-serif match)
- Body fallback    → DM Sans (Google Fonts — balanced grotesque)
